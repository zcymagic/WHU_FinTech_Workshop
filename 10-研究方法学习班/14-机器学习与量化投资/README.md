# 机器学习与量化投资(Machine Learning and Algorithmic Trading)

本学习班着重学习基于机器学习的量化投资系统的全貌，并采用Python实现具体的交易系统。基本上每一章都是一套完整的交易系统或论文实现。需要有一定的Python编程基础和机器学习基础，可以参考其他的学习班。
